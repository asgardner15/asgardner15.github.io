# Project 03 Scope Statement: Beginner's Guide to American Mahjongg Website

## Project Title
A Beginner's Guide to American Mahjongg

## Stakeholders
- Project Manager: Amy Gardner
- Instructor: Cheri Whalen
- Users: People interested in learning American Mahjongg
_______________________________________________________________________________
## Project Purpose
The purpose of this project is to create a simple, user-friendly website that introduces beginners to the game of American Mahjongg. 
The game can be overwhelming to new players due to the number of tiles, rules, and the annual card system. This website will simplify those concepts and provide a clear starting point for learning the game.
_______________________________________________________________________________
## Project Description
This project involves building a simple, user-friendly two-page website prototype using HTML and CSS that explains the basics of American Mahjongg. The website will include a homepage and an about page, both styled with a single CSS file.

The homepage will provide an overview of the game, including an introduction of what American Mahjongg is and why it is fun to play. The about page will focus on basic gameplay, including setup, tile types, and a brief explanation of how a round is played.

The design will be clean and easy-to-read, with simple page navigation. The focus of the project will be on structure and clarity.
________________________________________________________________________________
## Desired Results
The completed project will include:
1. A functioning two-page website.
2. An index.html homepage.
3. An about.html page linked from the homepage.
4. A single, external stylesheet (style.css) that applies consistent styling to both HTML pages.
5. A public GitHub repository with a live website.
6. A documentation folder containing:   
    - scope.md
    - plan.md
    - retrospective.md

## Success Criteria 
The project will be considered successful if:

## Exclusions
The project will not include:

- Advanced game strategy.
- Copies of any of the types of American Mahjongg cards.
- Details of how to play other types of Mahjong.
- Interactive features.
- Advanced functionality.
- Animations.
